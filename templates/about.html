Data Journalism Project: Getting around NYC

My Data Journalism Project

  • In my project, I am planning to explore the various ways that New Yorkers move around the city, and to see if I notice trends in the way that people in different boroughs move around. My dataset contains a record of the number of bicycles and motor vehicles, as well as the amount of times people use public transportation or walk in different neighborhoods across NYC. My dataset comes from nyc.gov database of Environment and Health Data.

My Data Cleaning Process

  1. Delete all rows that do not contain data for 2017-2021 (all rows past row 305)
  2. Within the 2017-2021 data, only keep data whose geotag is either UHF42 or borough or citywide
  3. Delete the GeoID and GeoRank columns
  4. Change column titles so as to remove spaces, parenthesis, commas
  5. Use google sheets “custom number formats” to remove commas in numbers with more than three digits. It is only necessary to apply this command to the “number” columns, and not the “percent” columns(removing these special characters can simplify converting the csv file to the json file)
  6. After manually cleaning the data using google sheets, download the data as a csv file and then reformat the data into a json file using a python data pipeline
